# Database-Analytics

Database Analytics coursework project based on the NorthStar Urban Mobility and Logistics dataset.

This project uses SQL in R, Python, and MongoDB Atlas to analyse delivery operations, operational performance, customer service issues, routing efficiency, and logistics reliability.

## Notebooks

### 01_python_setup.ipynb
Data loading, preprocessing, cleaning, and dataset integration.

### 02_sql_r_analysis.ipynb
SQL queries and R visualisations for delivery status, hub performance, driver analysis, complaints, and operational insights.

### 03_python_analysis.ipynb
Python statistical analysis, feature engineering, pairplots, correlation analysis, and operational visualisations.

### 04_mongodb_atlas.ipynb
MongoDB Atlas implementation including CRUD operations, aggregation pipelines, nested document schemas, and query optimisation using indexes.
